Prerequisites\n\n- Node.js 18+\n- .NET 8.0 SDK\n- SQL Server\n- Git\n\n### Installation\n\n1. **Clone Repository**\n   ```bash\n   git clone https://github.com/Mostafa-SAID7/Naar-Noor.git\n   cd Naar-Noor\n   ```\n\n2. **Backend Setup**\n   ```bash\n   cd api-server\n   dotnet restore\n   dotnet run --project src/NaarNoor.API/NaarNoor.API.csproj\n   ```\n\n3. **Frontend Setup** (New Terminal)\n   ```bash\n   cd naar-noor\n   npm install\n   npm run dev\n   ```\n\n### Verify\n\n- Frontend: http://localhost:4200\n- Backend: http://localhost:5000\n- Swagger: http://localhost:5000/swagger\n\n---\n\n## 📡 API Endpoints\n\n### Chefs\n- `GET /api/chefs` - Get all chefs\n\n### Menu\n- `GET /api/menu` - Get all menu items\n\n### Reservations\n- `GET /api/reservations` - Get all reservations\n- `POST /api/reservations` - Create reservation\n\n### Reviews\n- `GET /api/reviews` - Get approved reviews\n\n### Contact\n- `POST /api/contact` - Submit contact inquiry\n\n### Health\n- `GET /health` - Health check\n\n**Full API Documentation:** http://localhost:5000/swagger\n\n---\n\n## 🧪 Testing\n\n### Backend Tests\n```bash\ncd api-server\ndotnet
